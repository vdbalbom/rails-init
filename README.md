# rails-init

## Dependences

- docker
- docker-compose
- make

### To Build
```[bash]
$ make build
```

### To Test
```[bash]
$ make run-test
```

### To Run in DEV Environment
```[bash]
$ make run-dev
```

### To Run in PROD Environment
```[bash]
$ make run-prod
```

### To Run in PROD Environment - Detached
```[bash]
$ make run-prod-detached
```
